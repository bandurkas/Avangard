## Фаза 2: Боковая панель детализации ТС в смене (Vehicle Details Drawer)
- [x] **2.1. Создание выезжающей справа панели (Slide-over Drawer).**
  - Разработать красивую панель с анимацией выезда справа, закрывающуюся по клику на «крестик» или вне панели.
- [x] **2.2. Заполнение панели детальными данными ТС в смене.**
  - **Заголовок:** Модель ТС, Госномер, VIN, статус.
  - **Водитель:** Карточка активного водителя (ФИО, статус, аватар).
  - **Текущая смена:** 
    - Время старта и продолжительность смены (расчет в часах/минутах).
    - Пройденный путь (км).
    - Скорость и зажигание (В движении / Стоит, Двигатель заведен / Заглушен).
  - **Топливный бак:** Прогресс-бар остатка топлива в процентах и литрах с предупреждением при низком уровне (<20%).
  - **Лог событий:** Индивидуальная лента событий по данному ТС в этой смене (запуски, остановки, предупреждения о перерасходе топлива).

## Фаза 3: Модуль «Расчеты и Аналитика выплат» (Smart Payroll Tab)
- [x] **3.1. Навигация и интеграция вкладки.**
  - Добавить кнопку-вкладку «Расчеты (Payroll)» с иконкой `Wallet` в главное меню навигации.
  - Настроить отображение кнопки «Добавить ТС/Водителя» только на соответствующих вкладках.
- [x] **3.2. Аналитические карточки ФОТ (Payroll Overview).**
  - Разработать 4 карточки аналитики сверху:
    - Общий ФОТ (выплачено + к выплате)
    - Средний доход водителя в час
    - Сверхурочные часы (Overtime)
    - Корректировки (Премии за экономию топлива / Штрафы за пережог)
- [x] **3.3. Интерактивная таблица расчета выплат (Billing Spreadsheet).**
  - Таблица со столбцами: Водитель, ТС на смене, Отработано (часы в пути), Простой (часы ожидания), Ставка (руб/час), Коррекция ГСМ (₽) (динамически изменяемые), Итого к выплате.
  - Формула расчета: `(часы_в_пути * ставка) + (часы_простоя * ставка * 0.5) + бонус - штраф`.
  - Статусы выплат: «Выплачено» (зеленый), «В обработке» (оранжевый), «К выплате» (синий).
  - Кнопка действия «Выплатить» с красивой 1.2-секундной анимацией перехода статуса.
